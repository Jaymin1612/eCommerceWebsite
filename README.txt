BestDeal website is an eCommerce website where you can purchase televisions, tablets, phones and laptops.

There are 3 users: customer, retailer and sales person.

Customers are able to add multiple items to cart to purchase, update and delete orders from cart and place the order by giving credit card information. Once the order is placed, an order number generates and user can track the order and if he wants, cancel the order. 

A retailer is responsible for adding, updating information and deleting items to the website.

A sales person can add/delete customer accounts.
